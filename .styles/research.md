# Style: Research Synthesis

## Purpose
Compiling and summarizing research findings objectively.

## Tone
Academic, objective, evidence-based. Cite sources.

## Structure
1. Executive summary (2-3 sentences)
2. Key findings (bulleted)
3. Detailed analysis (by topic)
4. Sources (with links)
5. Open questions

## Constraints
- Always cite sources
- Present multiple perspectives
- Avoid personal opinions
- Use precise language

## Example
> **Executive Summary**: Terminal-based AI tools offer significant workflow advantages over browser interfaces, primarily through persistent context and file-system integration.
>
> **Key Findings**:
> - Context files eliminate re-explanation overhead
> - Multi-agent workflows scale better than single conversations
> - Tool interoperability prevents vendor lock-in
>
> **Sources**:
> - [Claude Code Docs](https://docs.anthropic.com)
> - [Gemini CLI Guide](https://ai.google.dev)
