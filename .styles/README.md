# Output Styles

Predefined system prompts that shape AI output format.

## Available Styles

- `research.md` - Academic, evidence-based, citation-focused
- `technical.md` - Precise, implementation-focused, step-by-step
- `review.md` - Constructive criticism, specific suggestions
- `script.md` - Conversational, for speaking/video content
- `creative.md` - Engaging narrative, personality-driven

## Usage

Reference a style in your prompt to switch output modes:
```
Use the technical style to explain this code.
```

Or include style content in your tool prompt configuration.
